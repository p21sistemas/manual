# Remessa

O conteúdo do arquivo foi desenvolvido com base no [Layout](http://manual.crabr.com.br/manual/?p=636) XML da FEBRABAN. O arquivo pode conter mais de uma **remessa**, agrupadas uma abaixo da outra, desde que esteja em conformidade com o padrão.
